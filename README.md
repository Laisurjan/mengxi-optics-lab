# 曲面鏡與透鏡成像 3D 互動實驗室

以《夢溪筆談》〈古人鑄鑑〉為核心的光學成像教學網頁，讓學生透過 3D 模型與滑桿操作，
理解凹面鏡、凸面鏡、凸透鏡、凹透鏡的成像規律，並對照課文文句與現實應用。

## 核心課文（沈括〈古人鑄鑑〉）

> 古人鑄鑑，鑑大則平，鑑小則凸。凡鑑窪則照人面大，凸則照人面小。
> 小鑑不能全現人面，故令微凸，收人面令小，則鑑雖小而能全納人面。
> 仍復量鑑之小大，增損高下，常令人面與鑑大小相若。
> 此工之巧智，後人不能造……此師曠所以傷知音也。

- **鑑窪（凹面鏡）則照人面大** → 成像放大
- **凸（凸面鏡）則照人面小、能全納人面** → 縮小、視野廣，小鏡也能照進整張臉

## 功能

- 🪞 四種光學元件：凹面鏡、凸面鏡、凸透鏡、凹透鏡
- 🌸 可選成像物體：小花、小熊、蠟燭、箭頭
- 🎚️ 滑桿操作：物距、焦距、物高
- 📐 即時顯示：實像/虛像、正立/倒立、放大率、像距，並說明「為什麼倒立／正立」
- 📖 課文對照：依操作即時螢光標出對應字句
- 🌍 現實應用：後視鏡、化妝鏡、放大鏡、相機、近視眼鏡等
- ▶️ 自動演示：物體由近到遠來回移動，展示成像完整變化
- 🧊 3D 視角：可拖曳旋轉、滾輪縮放，並有「課本平視圖／3D 立體視角」切換

## 使用方式

直接用瀏覽器（建議 Chrome / Edge）開啟 `optics.html` 即可，無需安裝。
3D 由 [three.js](https://threejs.org/) 提供（透過 CDN 載入，需連網）。
